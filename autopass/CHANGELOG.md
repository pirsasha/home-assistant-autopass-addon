# История изменений

## 0.8.3

- журнал показывает исправленный номер после позиционной OCR-нормализации;
- исходное значение Frigate остаётся доступно в журнале для диагностики;
- `K6S60B151` отображается как корректный номер `K656OB151`.

## 0.8.2

- машинные API теперь требуют настроенный `X-API-Key`;
- загрузка снимков ограничена хостом и портом настроенного Frigate;
- HTTP-редиректы при загрузке снимков запрещены.

## 0.8.1

- исправлено распознавание `0` как буквы `О` в буквенной позиции номера РФ;
- события ниже порога точности теперь видны в журнале как отказ.

## 0.8.0

- офлайн-лицензирование Ed25519;
- уникальный ID установки;
- демонстрационный и льготный периоды;
- аудит активации лицензии;
- светлая, тёмная и системная темы;
- резервные копии и периодическая очистка.
