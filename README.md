# Автопропуск для Home Assistant

Репозиторий установки коммерческого Home Assistant add-on для управления
автомобильными пропусками, распознавания номеров и открытия ворот.

## Установка

1. Откройте **Настройки → Дополнения → Магазин дополнений**.
2. Откройте меню **Репозитории**.
3. Добавьте:

   `https://github.com/pirsasha/home-assistant-autopass-addon`

4. Установите add-on **Автопропуск**.
5. Откройте add-on, скопируйте ID установки и передайте поставщику лицензии.
6. Загрузите полученный JSON-файл в разделе **Настройки → Лицензия**.

Исходный код приложения не публикуется в этом репозитории. Home Assistant
скачивает готовый контейнер из GitHub Container Registry.
